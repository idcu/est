## 快速开始

### 编译项目
```bash
mvn clean install -DskipTests
```

### 运行测试
```bash
# 使用Maven
mvn test

# 或使用自定义测试运行器
# 先编译
javac RunESTTests.java
# 再运行
java RunESTTests
```

### 启动est-admin
```bash
cd est-app/est-admin

# Windows
run-admin.bat

# Linux/Mac
./run-admin.sh
```

访问：http://localhost:8080/admin  
默认账号：admin / admin123
