## 使用示例

### 基础使用

```java
import ltd.idcu.est.[模块名].api.[接口];
import ltd.idcu.est.[模块名].impl.[工厂类];

public class Example {
    public static void main(String[] args) {
        [接口] instance = [工厂类].new[实现]();
        
        instance.[方法]("[参数]");
        [返回类型] result = instance.[方法2]();
    }
}
```

### 高级使用

```java
import ltd.idcu.est.[模块名].api.[接口];
import ltd.idcu.est.[模块名].impl.[工厂类];

public class AdvancedExample {
    public static void main(String[] args) {
        [接口] instance = [工厂类].<[泛型类型]>builder()
            .[配置方法1]([值1])
            .[配置方法2]([值2])
            .build();
        
        instance.[方法1]();
        instance.[方法2]();
    }
}
```

### 与其他模块集成

```java
import ltd.idcu.est.[其他模块].api.[其他接口];
import ltd.idcu.est.[模块名].api.[接口];
import ltd.idcu.est.[模块名].impl.[工厂类];

public class IntegrationExample {
    public static void main(String[] args) {
        [接口] instance = [工厂类].new[实现]();
        [其他接口] other = [其他工厂类].new[其他实现]();
        
        instance.[方法](other);
    }
}
```
