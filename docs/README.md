# EST Framework Documentation

Welcome to the EST Framework documentation!

## Overview

EST is a modern, zero-dependency Java framework with a progressive module structure.

## Architecture

The framework follows a clean, layered architecture:

```
est-examples (示例层)
    ├── est-tools (工具层)
    ├── est-app (应用层)
    ├── est-modules (模块层)
    ├── est-foundation (基础设施)
    ├── est-data-group (数据访问)
    ├── est-security-group (安全权限)
    ├── est-integration-group (消息集成)
    ├── est-web-group (Web框架)
    ├── est-ai-suite (AI套件)
    ├── est-microservices (微服务)
    └── est-extensions (扩展功能)
    ├── est-core (核心层)
    └── est-base (基础层)
```
