# Examples

This section contains practical examples for using the EST Framework.

## Example Projects

- **est-examples-basic**: Basic framework usage
- **est-examples-web**: Web application examples
- **est-examples-ai**: AI and LLM integration examples
- **est-examples-features**: Feature module examples
- **est-examples-advanced**: Advanced usage patterns
- **est-examples-graalvm**: GraalVM native image examples
- **est-examples-microservices**: Microservices examples

## Running Examples

Check the individual example directories for run scripts.
